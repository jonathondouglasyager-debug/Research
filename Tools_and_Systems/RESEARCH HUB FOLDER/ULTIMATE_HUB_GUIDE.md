## ğŸ› TROUBLESHOOTING

### **"Files won't drop"**

**Check:**
- Using Chrome/Firefox/Edge (not IE)
- File types: PDF, CSV, JPG, PNG
- File not corrupted
- Try click upload instead

### **"OCR is slow/fails"**

**Solutions:**
- OCR takes 5-10 sec per page (normal)
- Large PDFs = longer processing
- Poor image quality = worse results
- Try higher resolution image
- Check browser console (F12) for errors

### **"Claude not responding"**

**Causes & fixes:**
- API might be unavailable (wait & retry)
- Network issue (check connection)
- Rate limit (wait 30 seconds)
- Browser blocking API (check console)

### **"CSV not loading"**

**Check:**
- File is actually CSV (not Excel .xlsx)
- Proper CSV format (comma-separated)
- No special characters in headers
- Try opening in Excel first to verify

### **"Processing stuck"**

**Solutions:**
- Check browser console (F12) for errors
- Refresh page and try again
- Try smaller file first
- Clear browser cache
