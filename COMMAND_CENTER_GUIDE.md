## TROUBLESHOOTING

### Links Don't Work
**Problem:** Click a link, nothing happens
**Solution:**
- Make sure file paths are correct
- File may have moved - check `C:\Users\jonat\Documents\Research\`
- Some browsers block file:/// links - try different browser

### Notes Don't Save
**Problem:** Notes disappear after closing browser
**Solution:**
- Notes are saved in browser's localStorage
- If you clear browser data, notes are deleted
- Use Chrome/Edge for best compatibility
- Don't use Incognito/Private mode

### Search Doesn't Find Everything
**Problem:** Search misses files you know exist
**Solution:**
- Search only looks at visible text and keywords
- File content isn't searched (only names)
- Use File Explorer search for deep file content search

### Page Looks Weird
**Problem:** Formatting is broken
**Solution:**
- Press `Ctrl+F5` to hard refresh
- Try different browser (Chrome, Edge, Firefox)
- Make sure JavaScript is enabled

**Q: Does this replace my Master Research Hub?**
A: No - they're complementary. This is for navigation/overview, Hub is for analysis/processing.

**Q: Can I change the colors?**

**Q: What if I add a new investigation?**
